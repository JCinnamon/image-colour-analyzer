Batch Image Colour Analysis

Analyse colour clusters across multiple images and download colour data

Image URL requirements
  • Direct image links only — URL must end in .jpg, .jpeg, or .png
  • Publicly accessible — no login or authentication required
  • Use HTTPS where possible for better compatibility
e.g. https://i.imgur.com/example.jpg  ·  https://example.com/photos/image.png

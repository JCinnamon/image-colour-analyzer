# image-colour-analyzer — Image Colour Analyzer

A browser-based tool for analyzing colour properties across multiple images at once.
Paste image URLs, configure clustering, and download results as a CSV file.

**[Live →](https://www.jonathancinnamon.com/image-colour-analyzer/)**

---

## Features

- Accepts lists of image URLs (one per line)
- Configurable analysis precision and number of colour clusters
- Produces a per-image colour summary with hex values, cluster membership, and proportions
- Export results as CSV for downstream analysis

## Usage

1. Paste image URLs into the input box — one URL per line
2. Set analysis precision and number of colour clusters
3. Run the analysis and review the colour summaries
4. Download the CSV for use in R, Python, or a spreadsheet

Images must be publicly accessible URLs. See the in-tool notes on CORS requirements.

Credit: This tool accesses the Image Color Summarizer API by Martin Krzywinski.
